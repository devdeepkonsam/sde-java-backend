## Topics Overview

### Arrays & Hashing (Week 1)
Master the fundamentals of array manipulation and hash table usage.

**Key Patterns**:
- Frequency counting with HashMap
- Using HashSet for O(1) lookups
- Array transformations

**Problems**: Contains Duplicate, Valid Anagram, Two Sum, Group Anagrams, Top K Frequent Elements, Product of Array Except Self, Valid Sudoku, Encode and Decode Strings, Longest Consecutive Sequence, 3Sum

---

### Two Pointers (Week 2)
Learn to optimize array problems using two-pointer technique.

**Key Patterns**:
- Left and right pointers moving towards each other
- Fast and slow pointers
- Sorted array optimization

**Problems**: Valid Palindrome, Two Sum II, 3Sum, Container with Most Water, Trapping Rain Water

---

### Sliding Window (Week 3)
Optimize substring/subarray problems with sliding window.

**Key Patterns**:
- Fixed-size window
- Variable-size window
- Window with constraints

**Problems**: Best Time to Buy and Sell Stock, Longest Substring Without Repeating Characters, Longest Repeating Character Replacement, Permutation in String, Minimum Window Substring, Sliding Window Maximum

---

### Stack (Week 4)
Use LIFO data structure for parentheses, monotonic sequences.

**Key Patterns**:
- Matching pairs (parentheses)
- Monotonic stack (next greater/smaller element)
- Expression evaluation

**Problems**: Valid Parentheses, Min Stack, Evaluate Reverse Polish Notation, Generate Parentheses, Daily Temperatures, Car Fleet, Largest Rectangle in Histogram

---

## Study Tips

1. **Pattern Recognition**: Most problems follow 3-4 common patterns
2. **Draw It Out**: Visualize pointers and windows on paper
3. **Edge Cases**: Empty arrays, single elements, duplicates
4. **Optimization**: Start with brute force, then optimize
