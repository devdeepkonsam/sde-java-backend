Inheritance:

## What is Inheritance?
Inheritance is key pillar of OOPs where a new class(child/sub) inherite the methods ,datas
from the existing parent class(superclass/parent)
Class A Parent
  ^
  |
Class B Child


## Why Inheritance?
1. Code Reusability: We can reuse the code of parent class in child class.
2. Method Overriding: We can override the method of parent class in child class to provide specific implementation.
3. Polymorphism: We can use parent class reference to refer to child class object, which is called upcasting. This allows us to achieve runtime polymorphism.
4. Extensibility: We can extend the functionality of parent class by adding new methods in child class.

## Types of Inheritance
1. Single Inheritance: A child class inherits from a single parent class.
2. Multilevel Inheritance: A child class inherits from a parent class, which in turn inherits from another parent class.
3. Hierarchical Inheritance: Multiple child classes inherit from a single parent class.
4. Multiple Inheritance: A child class inherits from multiple parent classes (not supported in Java, but can be achieved using interfaces).
5. Hybrid Inheritance: A combination of two or more types of inheritance.
