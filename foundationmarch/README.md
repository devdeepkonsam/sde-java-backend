# Phase 1: Foundation - March

**Focus**: Java Fundamentals, OOP Principles, and Basic DSA

## Overview
This phase covers the core Java concepts that every backend engineer must master. Each week includes:
- **DSA Practice**: 5-10 LeetCode problems from NeetCode 150
- **Java Tasks**: Hands-on coding exercises demonstrating OOP principles
- **Theory**: Conceptual understanding of JVM, memory management, and design principles

## Weekly Structure

### Week 1: Encapsulation & Arrays/Hashing
Focus on data hiding, getters/setters, and array-based algorithms.

### Week 2: Inheritance & Two Pointers
Explore code reuse through inheritance and master two-pointer techniques.

### Week 3: Abstraction & Sliding Window
Learn interface design and sliding window pattern for optimization.

### Week 4: Modern Java & SQL Basics
Apply functional programming (Streams, Lambdas) and learn SQL fundamentals.
Focus week 4 on exception handling, JDBC, and SQL basics first.
Defer multithreading and concurrency to a later phase so the pace stays steady.
Add annotation and reflection basics here so week 5 starts from a familiar base.

## Learning Tips

1. **Write Code Daily**: Consistency beats intensity
2. **Understand, Don't Memorize**: Focus on "why" over "what"
3. **Document Your Learning**: Use markdown files in each week folder
4. **Test Your Code**: Write simple test cases for each exercise
