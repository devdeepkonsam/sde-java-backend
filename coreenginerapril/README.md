# Phase 2: Core Engineer - April

**Focus**: Spring Boot Framework, REST APIs, and Database Integration

## Overview
This phase transitions from core Java to enterprise application development. You'll build a real-world Spring Boot project that evolves weekly, incorporating:
- **Spring Framework**: IoC, DI, Bean lifecycle
- **REST APIs**: Building RESTful endpoints with proper HTTP semantics
- **Database Integration**: JPA/Hibernate with MySQL
- **Entity Relationships**: Modeling complex data structures

## Weekly Structure

### Week 5: IoC & Dependency Injection
Set up your first Spring Boot project and understand the core container concepts.

### Week 6: Building REST APIs
Create your first REST endpoints with proper request/response handling.

### Week 7: Database Persistence
Connect to MySQL and implement JPA entities with CRUD operations.

### Week 8: Entity Relationships
Model complex relationships like User-Post (@OneToMany, @ManyToOne).

---

## Project Structure
The `backend-project/` folder is your single, evolving codebase. By the end of April, it will have:
- ✅ REST Controllers
- ✅ Service Layer with interfaces
- ✅ JPA Repositories
- ✅ MySQL database integration
- ✅ Proper error handling

---

## Learning Tips

1. **One Project, Many Features**: Don't create new projects weekly—evolve the same one
2. **Understand Annotations**: @Component, @Service, @Repository, @Autowired
3. **Test with Postman**: Keep a collection of your API requests
4. **Read Stack Traces**: Learn to debug Spring Boot errors
